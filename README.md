# Space Invaders - Unity Project

## Descripción
Este proyecto es una recreación del clásico videojuego **Space Invaders** desarrollada en Unity.  
El objetivo del juego es eliminar a los enemigos mientras evitas sus disparos utilizando los escudos para protegerte.

Este proyecto fue desarrollado como parte de un trabajo académico en equipo.

---

## Características
- Movimiento del jugador izquierda/derecha
- Disparo de proyectiles
- Enemigos que patrullan y disparan automáticamente
- Sistema de puntuación
- Temporizador de partida
- Escudos defensivos
- Menú principal

---

## Controles

| Tecla | Acción |
|------|------|
A / ← | Mover izquierda |
D / → | Mover derecha |
Espacio | Disparar |

---

## Tecnologías utilizadas
- Unity (Motor de videojuegos)
- C#
- TextMeshPro
- Sistema de físicas 2D

---

## Estructura del proyecto

Scripts principales:

- `shooting.cs` → Control del jugador
- `ENEMIES.cs` → Comportamiento de los enemigos
- `Projectile.cs` → Lógica de las balas
- `SCORE_MANAGERBALL.cs` → Sistema de puntuación y temporizador
